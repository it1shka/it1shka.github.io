Web Dev Job :)

There are a lot of jobs to choose from in our modern world. My choice is Web Developer!

What is Web Development?

I think most people from our English group want to become software developers in their future. At least, according to the survey we took on the previous lesson :) Programming is not a strictly defined area. There are a lot of directions in programming. You can be anything: you can design robots, you can write games, you can implement your own low-level systems.

One of such directions is Web Development! In simple terms, it's a job where you create websites.

Is it difficult?

Web Development is tough beacuse there are too many things you need to know! To create a fully-fledged website you need to know both how to create user interfaces and servers.

Web Dev = Front-end + Back-end

The first part of your website is front-end. This is all about creating and coloring buttons, panels, navbars...

To be successful in front-end you need to deeply know HTML and CSS + JavaScript as well as any of JS libraries.

Creating layout of your app is a very exciting process that touches a lot of aspects from design and requires a lot of imagination

All the fancy styling and interaction is front-end!

What is back-end?

Usually it is much harder to implement backend for your website than to deal with widgets. When you are a back-end developer you need to deal with servers and databases, and it's pretty complex stuff!

And if on the front-end part you will use JavaScript as your primary language, on backend it's up to you which technology to choose!

Why did I chose this job?

  • It is well-paid
  • It is extremely interesting for me
  • I already have some skills
  • There is huge demand in IT specialist on the global market
